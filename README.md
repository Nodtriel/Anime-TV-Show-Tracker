# Anime & Show Tracker - Chrome Extension

A powerful Chrome extension to track your favorite anime and TV shows with ease.

## Features

✅ **Separate Tracking** - Keep anime and TV shows separate  
✅ **Full View Mode** - Dedicated full-page interface for managing your collection  
✅ **Episode Tracking** - Track current season and episode  
✅ **Poster Images** - Add URL-based poster images for each show  
✅ **Watch Links** - Store links to where you watch each show  
✅ **Notes** - Add personal notes to each show  
✅ **Filter & Search** - Filter by All, Anime, or TV Shows  
✅ **Local Storage** - All data stored locally in Chrome  

## Installation

1. **Open Chrome Extensions Page**
   - Go to `chrome://extensions/`
   - Or: Menu → More Tools → Extensions

2. **Enable Developer Mode**
   - Toggle "Developer mode" in the top-right corner

3. **Load the Extension**
   - Click "Load unpacked"
   - Navigate to and select the "Anime and Show Tracker" folder
   - The extension will appear in your extensions list

4. **Pin the Extension** (Optional)
   - Click the extensions icon in your toolbar
   - Click the pin icon next to "Anime & Show Tracker" to keep it visible

## Usage

### Quick Add (Popup)
- Click the extension icon to open the quick menu
- Use "Open Full View" or "Quick Add Show" to access the full interface

### Full View (Main Interface)
- Click "Open Full View" to open the complete tracker in a new tab
- Use **"+ Add New"** to create a new show entry
- **Filter shows** using the top buttons:
  - All: Show everything
  - Anime: Only anime
  - TV Shows: Only TV shows

### Adding a Show
1. Click "+ Add New"
2. Fill in the details:
   - **Show/Anime Name**: Title of the show
   - **Type**: Select Anime or TV Show
   - **Current Season**: Season number
   - **Current Episode**: Episode number
   - **Watch Link**: URL to where you watch it (optional)
   - **Poster Image**: URL to the poster image (optional)
   - **Notes**: Any notes about the show (optional)
3. Click "Save Show"

### Editing a Show
- Click the **✏️ Edit** button on any show card
- Modify the details as needed
- Click "Save Show" to update

### Deleting a Show
- Click the **🗑️ Delete** button on any show card
- Confirm the deletion

### Managing Images
- Enter a URL to a poster image
- The image preview will display as you type
- Images can be from any URL (IMDb, MyAnimeList, etc.)

## Data Storage
- All your data is stored locally in Chrome's storage
- Data persists across browser sessions
- Data is NOT synced across devices or browsers

## Tips
- Use high-quality image URLs for better visuals
- Add notes to remember where you left off or your thoughts
- Keep watch links handy for quick access to streaming services
- Use the filter buttons to organize by content type

## Support
If you encounter any issues:
1. Clear Chrome cache and reload the extension
2. Check the browser console for errors (F12 → Console)
3. Reinstall the extension if problems persist

Enjoy tracking your shows and anime! 📺🎌
